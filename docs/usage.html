更新通知を受け取るには?

このBOTで受け取れる通知は二種類あります。
一つはBOTの更新情報で、もう一つはMilkChoco運営の更新情報です。
BOTの更新情報を受け取るには m!follow コマンド、
運営の更新情報を受け取るには m!notice コマンドを送信してください。
これにより、あなたのサーバーで更新情報をいち早く受け取れます!

グローバルチャットとは?

グローバルチャットとは他のサーバーに居る人とBOTを介してお話しできるサービスです。
グローバルチャットでは他のサーバーから届いたメッセージはBOTと表示されますが、中身は人間です(笑)
グローバルチャットを設定するには、設定したいチャンネルで m!global join と送信してください。
ただし、その時点からそのチャンネルで送信したメッセージは他のサーバーにも表示されて、 また、他のサーバーのメッセージがそのチャンネルに表示されることに気を付けてください!
専用チャンネルを作成しておくことをおすすめします。
グローバルチャットに設定したチャンネルでは、コマンドが使用できなくなります。

グローバルチャットでの禁止事項

当サービスでのグローバルチャットでは以下の行為を禁止しています。
違反した場合、予告なくロック、ミュート処置をとることがあります。

  1. 成人向けコンテンツまたはグロテスクな表現を含むコンテンツの送信
  2. 誹謗中傷を含むコンテンツの送信
  3. サーバー招待URL等を送信して宣伝する行為
  4. 短い間隔で何度も文章を送信し、負荷をかける行為
  5. BOTのコマンドを送信する行為
  6. その他当チームが極めて不適切だと判断するコンテンツ

グローバルチャットでのロックされた場合の解除方法

グローバルチャットでのアカウントロックを受けた場合、以下の手順で解除することができます。

  1. グローバルチャットでの禁止事項に掲載されている禁止事項を再確認する
  2. BOTのDMに unlock と送信する
以上でアカウントロックを解除できます。解除後に違反行為をすると、ミュートされる確率が高くなりますので、お気を付けください。

BOTの招待時の権限について

BOTを招待する際は、基本的にそのBOT用の役職が自動的に作成されます。
その権限はあとからも変更することができます。
当BOTではリクエストに応じて機能を追加する可能性があり、権限に関して詳しい人には問題ないと思われますが、 権限設定がよくわからないという方がいらっしゃることを考慮して、標準権限を管理者に設定しています。
先ほど申し上げたように、招待時の権限は招待後に変更することもできるので、必要に応じて設定してください。
