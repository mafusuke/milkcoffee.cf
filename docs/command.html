コマンド一覧

MilkCoffeeBOTのすべてのコマンドはこちらで確認できます。
全くどう使うのか分からない!という方はこちらを先に確認してください。
以下のカテゴリ名をクリックすると、そのカテゴリの場所にジャンプします。
コマンド名をクリックすると、そのコマンドの説明文が表示されます。

  • GlobalChat
  • Notify
  • Costume
  • Information

GlobalChat

他のサーバーに居る人と、設定したチャンネルでお話しできるよ!

global [サブコマンド]

グローバルチャットに関するコマンドだよ!
グローバルチャット設定をするためには、BOTが manage_webhook(webhookを管理) の権限を持ってて、コマンドの実行者が manage_channel(チャンネルの管理) 権限を持っている必要があるよ!

global join (チャンネル)

グローバルチャットに接続するよ!。チャンネルを指定しなかったら、コマンドが実行されたチャンネルに設定するよ!。

m!global join ... コマンドを打ったチャンネルをグローバルチャットに接続します。
m!global join #チャンネル ... 指定したチャンネルをグローバルチャットに接続します。

global leave [チャンネル]

指定したチャンネルをグローバルチャットから切断するよ!。(グローバルチャットに接続されているチャンネルではコマンドは実行できないから気を付けてね)

m!global leave #チャンネル ... 指定したチャンネルをグローバルチャットから切断します。

Notify

お知らせの設定するよ!

follow (チャンネル)

BOTのお知らせをあなたのサーバーのチャンネルにお届けするよ!チャンネルを指定しなかったら、コマンドを実行したチャンネルにお知らせするよ!

notice (チャンネル)

MilkChoco運営の更新情報をあなたのサーバーのチャンネルにお届けするよ!

unnotice (チャンネル)

MilkChoco運営の更新情報の受け取りを停止するよ!

ads

次広告が見れるまでの10分間のタイマーを設定するよ!広告見ようとはおもってるけど、忘れちゃう人向け。

Costume

装飾シミュレータを操作できるよ！好みの組合せを探そう！

set [装飾コード|各装飾の番号]

装飾コードまたは各装飾の番号で設定できるよ!.

この二つのコマンドは両方ミルクアサルト初期武器(装飾無し)になるよ!。
m!set 1o4s3k ... 装飾コード1o4s3kで設定
m!set 0 1 1 0 0 0 ... 各アイテムの番号で設定
装飾コードは他の人の装飾を真似する際に便利だよ!

show (保存番号|保存名称)

現在の装飾を表示できるよ!保存番号を指定したら、保存した作品の中から番号にあった作品を表示してあげる!

m!show ... 現在の装飾を表示
m!show 1 ... 1番目に保存された装飾を表示
m!show 無題1 ... 無題1という名前で保存された装飾を表示

load [保存番号|保存名称]

保存した作品を番号または名称で指定して、現在の作業場に読み込むよ!

m!load 1 ... 1番目に保存された作品を読み込む
m!load 無題1 ... 無題1という名前で保存された作品を読み込む

save (保存名称)

現在の装飾を保存できるよ!保存名称を指定しなかったら、'無題1'みたいな名前を自動でつけとくね!

m!save ... 作品を保存します(名前は自動で無題1のように付けられます)
m!save 新作品 ... 新作品という名前で作品を保存します

my (ページ)

保存した作品の一覧を表示できるよ!ページを指定しなかったら、1ページ目から表示するよ!でも、リアクションを押してページ移動もできるから心配しないでね!

略記: mylist

m!my ... 保存した作品集の1ページ目を表示します
m!my 2 ... 保存した作品集の2ページ目を表示します

delete [保存番号|保存名称]

保存した作品を番号または名称で指定して削除するよ!一度削除したらその作品は戻せないから気を付けてね!

略記: remove, del, rm

m!delete 1 ... 1番目に保存された作品を削除します
m!delete 旧作品... 旧作品という名前の作品を削除します

add [種類] [番号|名称]

アイテムを追加するよ!
1つ目の'種類'にはbase/character/weapon/head/body/back(詳しくはhelpコマンドの?リアクションを押して確認してね)のいずれかを指定して、
2つ目の'番号|名称'にはアイテムの名前または番号を指定してね!

m!add weapon AT ... ATという名前の武器を追加します
m!add head 1 ... 1番の頭装飾を追加します

add body [番号|名称]

体装飾を設定できるよ!

略記: d, bd, by

m!add body 5...番目の体装飾を設定します
m!add body n.s.suit ... n.s.suitという名前の体装飾を設定します

add character [番号|名称]

キャラクターを設定できるよ!。

略記: c, ch, char

m!add character 2 ... 2番目のキャラクターを設定します
m!add character air ... キャラクターをairに設定します

add back [番号|名称]

背中装飾を指定できるよ!

略記: b, bk, bc

m!add back 6...6番目の背中装飾を設定します
m!add back B.MOUSE ... B.MOUSEという名前の背中装飾を設定します

add weapon [番号|名称]

武器を設定できるよ!

略記: w, wp, weap

m!add weapon 3 ... 3番目の武器を設定します
m!add weapon spyra ... spyraを武器に設定します

add item [名称]

アイテムを追加できるよ!名前を教えてくれたら、全種類の中から探すからね!

略記: i

検索対象が全種類で広いから、思っているものと違うアイテムとマッチする可能性もあるよ> < また、全種類対応だから各種類のアイテム番号は使えないよ.。
m!add item myocat ... myocatという名前のアイテムを全種類から検索して追加します

add head [番号|名称]

頭装飾を設定できるよ!

略記: h, hd

m!add head 4 ... 4番目の頭装飾を設定します
m!add head M.CHIKEN ... M.CHIKENという名前の頭装飾を設定します

add base [番号|名称]

白黒を設定できるよ!。

略記: s, bs

m!add base 0 ... 0番目の色を設定します(白色)
m!add base choco ... chocoを設定します（黒色)

list [種類]

その種類のアイテム一覧を表示するよ!

m!list character ... キャラクターのリストを表示します
m!list weapon ... 武器のリストを表示します

list back

背中装飾のリストを表示するよ!

略記: b, bc, bk

m!list back ... 背中装飾のリストを表示します

list character

キャラクターのリストを表示するよ!

略記: c, ch, char

m!list weapon ... 武器のリストを表示します

list head

頭装飾のリストを表示するよ!

略記: h, hd

m!list head ... 頭装飾のリストを表示します

list base

白黒のリストを表示するよ!この場合は白と黒の二つしかないんだけどね💦

略記: s, bs

m!list base ... キャラ色のリストを表示します

list body

体装飾のリストを表示するよ!

略記: d, bd, by

m!list body ... 体装飾のリストを表示します

list weapon

武器のリストを表示するよ!

略記: w, wp, weap

m!list character ...キャラクターのリストを表示します

Information

色々な情報の設定をするよ!

invite

BOTの招待リンクを表示するよ!是非いろんなサーバーに招待してね!。

略記: inv

info

BOTに関する情報を表示するよ!。

略記: about

ping

BOTの反応速度を計測するよ!。

略記: pg
